Grow the Galaxy

Game Development / Lua / Roblox Studio / Sci-Fi Idle Farming Game

Project Summary

A cinematic survival farming sim built entirely in Roblox Studio, Grow the Galaxy drops players on the Moon following a catastrophic solar storm. With only a single alien seed and a curious nanobot sprout, players must establish life inside a fragile lunar biodome — and ultimately begin transforming dead space into thriving life.

Features
  • Custom Moon environment
  • Scripted transparent biodome
  • Idle/offline growth logic
  • Nanobot NPC interactions
  • Cinematic cutscenes
  • Core gameplay loop

This is an original property created by code+canvas as part of my creative prototyping and IP development slate.
